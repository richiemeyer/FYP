About Page

This is a sample web mapping project by Richard Meyer -
A student in TU856 Year 4.

  • The site is hosted by a DigitalOcean docker droplet and uses a domain from GoDaddy.
  • The domain is richardmeyer.xyz and utilizes https encryption using nginx and certbot
  • It is deployed as a progressive web app and can be installed on mobile or desktop
  • The idea of this web app is to promote safe driving by letting users record a journey and see how their speed compares to the speed limit on the current road their on
  • Information on speed limits is gathered from Overpass. This is a read-only API used to query geospatial information
